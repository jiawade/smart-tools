
# smart tools #

## Installation

#### Maven
````xml
<dependency>
  <groupId>io.github.jiawade</groupId>
  <artifactId>smart-tools</artifactId>
  <version>0.0.1</version>
</dependency>
````

#### Gradle
````gradle
compile 'io.github.jiawade:smart-tools:0.0.1'
````
